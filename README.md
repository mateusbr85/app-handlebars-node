
# Aplicação de estudo Node.js e Handlebars

Essa Aplicação foi criada para intuito de estudo das ferramentas node.js e handlebars, aonde pretendia criar uma pequena Aplicação de listagem e delete, tornando assim dinamica, utilizando padrões de rotas e também de models para o banco de dados, que foi construido com MySql.


## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/mateusbr85/app-handlebars-node.git
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run dev
```
